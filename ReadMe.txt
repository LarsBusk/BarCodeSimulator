The program simulates a barcode reader.
It will is possible to send single codes, or codes for each new sample.
When Randoom code is selected, it sends a code that is 'Start position' + 1 long.
The last digit is between 1 and 4 and can be used for automatic product switch.
It can be used on virtual machines with a serial port simulator from http://www.hhdsoftware.com/virtual-serial-ports
